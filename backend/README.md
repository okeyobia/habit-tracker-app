# FastAPI Backend for Habit Tracker

This backend provides API endpoints, authentication, and database models for the habit tracker app.

## Structure
- app/
  - main.py
  - models.py
  - schemas.py
  - crud.py
  - auth.py
  - database.py
  - dependencies.py
- requirements.txt

## Features
- JWT authentication
- User, Habit, HabitTracking models
- CRUD endpoints for habits and tracking
- Pydantic validation
